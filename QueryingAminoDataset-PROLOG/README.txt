AminoAcids.txt is a data set that the Prolog_Query.txt file includes a list of queries to search for particular Amino Acids. To test this, and compile it the following can be done:

Click on this link for an online IDE: https://swish.swi-prolog.org/

Alternatively, any Prolog IDE can be downloaded.

If you wish to view the answers of each query in the Prolog_Query.txt file, then you may view them in the Prolog_Answers.txt file
